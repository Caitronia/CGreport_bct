Simple Algorithm for the Bresenham Line Drawing Code

Input: Start point (x0, y0), End point (x1, y1)
Output: List of points forming the line

1. Calculate dx = |x1 - x0| and dy = |y1 - y0|
2. Determine step directions:
   sx = 1 if x1 > x0 else -1
   sy = 1 if y1 > y0 else -1
3. Initialize x = x0, y = y0
4. If dx >= dy:
   a. err = dx / 2
   b. Repeat dx + 1 times:
   i.  Add (x, y) to points
   ii. Move x by sx
   iii. err -= dy
   iv. If err < 0: move y by sy and err += dx
5. Else:
   a. err = dy / 2
   b. Repeat dy + 1 times:
   i.  Add (x, y) to points
   ii. Move y by sy
   iii. err -= dx
   iv. If err < 0: move x by sx and err += dy
6. Return the list of points
7. Plot points using a line with markers using matplotlib.
