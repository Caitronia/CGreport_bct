Step 1: Start

Input the starting point (x1, y1) and ending point (x2, y2).

Step 2: Calculate differences

dx = x2 - x1

dy = y2 - y1

Step 3: Determine the number of steps

steps = max(|dx|, |dy|)

Step 4: Calculate increments

x_inc = dx / steps

y_inc = dy / steps

Step 5: Initialize starting point

x = x1

y = y1

Initialize empty lists: x_points and y_points.

Step 6: Generate points

Repeat for steps + 1 times:

Append round(x) to x_points

Append round(y) to y_points

Update x = x + x_inc

Update y = y + y_inc

Step 7: Print coordinates

Print x_points and y_points.

Step 8: Plot the line

Use plt.plot(x_points, y_points) to draw the line.

Add title, labels, and grid.

Display the plot with plt.show().
